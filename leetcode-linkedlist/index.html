【21】合并两个有序链表

题目

解法

先新建一个链表，得到头结点指针和当前指针current，最开始current指向的是头结点。

对两个链表进行遍历，当有其中一个遍历完之后就退出循环。

每次遍历对两个节点进行比较，将值小的节点添加到结果中，即current.next = Math.min(l1,l2)，然后，小的节点对应的链表往后移动一个节点，大的节点对应的链表不动

同时current指针后移，即current = current.next

循环退出后，只需要将剩余的链表的节点直接添加到新的链表中即可得到最终的结果

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} l1
 * @param {ListNode} l2
 * @return {ListNode}
 */
var mergeTwoLists = function (l1, l2) {
    let link1 = l1, link2 = l2;
    let head = new ListNode('head')
    let current = head;
    while (link1 != null && link2 != null) {
        if (link1.val < link2.val) {
            current.next = link1;
            link1 = link1.next;
        } else {
            current.next = link2;
            link2 = link2.next
        }
        current = current.next;
    }
    if (link1) current.next = link1;
    else if (link2) current.next = link2;
    return head.next;
};

通过迭代，取相邻的两个链表合并，得到一个新的合并后的链表，然后用这一个新的链表去和下一个链表来合并，直到所有的链表合并完，得到最终的一个合并链表

 *     this.next = (next===undefined ? null : next)

/**
 * @param {ListNode[]} lists

var mergeKLists = function (lists) {
    if (lists.length === 0) return null;
    let targetList = lists[0];
    let index = 0;
    while (++index < lists.length) {
        targetList = merge(targetList, lists[index])
    }
    return targetList;
};
// 合并两个升序链表
const merge = function (list1, list2) {
    let l1 = list1;
    let l2 = list2;
    let head = new ListNode(-1);
    let current = head;
    while (l1 && l2) {
        if (l2.val < l1.val) {
            current.next = l2;
            l2 = l2.next;
        } else {
            current.next = l1;
            l1 = l1.next
        }
        current = current.next;
    }
    if (l1) current.next = l1;
    if (l2) current.next = l2;
    return head.next;

【面试题02.08】环路检测

题目

解法

先通过一个快指针fast和一个慢指针slow对链表进行遍历，快指针每次前进两个节点，慢指针每次前进一个节点

如果链表没有环，则快指针很快会遍历完，并且此时fast !== slow，因此可以直接返回 null

如果链表有环，则快指针和慢指针迟早会在某一个节点相遇，此时可以跳出遍历的循环。

判断出了有环后，还需要找出环的入口节点。

此时让快指针重新指向头结点fast = head（或者新建一个都一样，目的是要有一个指针从头结点开始遍历），慢指针依旧停在相遇的节点位置，然后头指针fast和慢指针slow都以相同的速度每次移动一个节点

根据数学推导可以知道，头指针fast和慢指针slow第一次相遇的位置恰好就是环的入口节点。

关于为什么恰好在环的入口相遇，可以查看Leetcode上题解中一个推导过程

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var detectCycle = function (head) {
    if (head == null || head.next == null) return null;
    let fast = slow = head;
    while (fast && fast.next) {
        fast = fast.next.next;
        slow = slow.next;
        if (fast === slow) break;
    }
    if (fast !== slow) return null;
    fast = head;
    while (fast !== slow) {
        fast = fast.next;
        slow = slow.next;
    }
    return fast;
};

【328】奇偶链表

题目

解法

用一个指针存储偶链表，对链表进行遍历，每次将奇节点的后一个节点添加到偶链表后面，然后在原链表中删掉这个节点 odd.next=odd.next.next，然后奇链表（原链表）遍历指针后移

最后循环完后，将偶链表接到奇链表后面即可。

/**
 * Definition for singly-linked list.
 * function ListNode(val, next) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.next = (next===undefined ? null : next)
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var oddEvenList = function (head) {
    let oddHead = head;
    let odd = oddHead;
    let evenHead = new ListNode('head');
    let even = evenHead;
    while (odd) {
        even.next = odd.next;
        even = even.next;
        if (!(even && even.next)) {
            break;
        }
        odd.next = odd.next.next;
        odd = odd.next;
    }
    odd && (odd.next = evenHead.next);
    return oddHead;
};

解法

解决问题的关键是将双向链表旋转九十度，看成是一颗二叉树，节点的child子链表指针就相当于是树中节点的左指针，节点的next指针相当于是树中节点的右指针

然后通过对树进行前序遍历（深度优先遍历）的顺序，就是扁平化后的结果。

在实现的时候，通过深度优先的思想，通过一个current指针，从头结点开始开始遍历，将current节点的子链表current.child进行扁平化，然后返回扁平化后的链表的尾节点，与current节点的下一个节点current.next进行连接

具体的解法描述可以查看LeetCode官方题解

/**
 * // Definition for a Node.
 * function Node(val,prev,next,child) {
 *    this.val = val;
 *    this.prev = prev;
 *    this.next = next;
 *    this.child = child;
 * };
 */

/**
 * @param {Node} head
 * @return {Node}
 */
var flatten = function (head) {
    if (head === null) return head;
    let current = new Node(-1, null, head, null);
    flattenDFS(current,head);
    current.next.prev = null;
    return current.next;
};
function flattenDFS(prev, current) {
    if (current === null) return prev;
    prev.next = current;
    current.prev = prev;
    let next = current.next;
    const tail = flattenDFS(current, current.child);
    current.child = null;
    return flattenDFS(tail, next)
}

【面试题02.02】返回倒数第K个节点

题目描述

题目链接

解法

方法1 数组存储

通过一个数组来存放每个节点的值，然后一直是遍历链表，直到链表遍历结束

然后返回arr[arr.length - k]就可以得到倒数第K个节点

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @param {number} k
 * @return {number}
 */
var kthToLast = function (head, k) {
    const arr = []
    let current = head;
    while (current) {
        arr.push(current.val);
        current = current.next;

    }
    return arr[arr.length - k]
};

时间复杂度：O(n)，

空间复杂度：O(n)

解法2 双指针

通过两个指针fast和slow进行遍历，先让fast指针往后遍历k个节点，然后fast和slow再同时往后遍历，相当于是用slow指针将fast指针前面相隔k个节点的节点存储下来

当fast节点遍历到链表末尾的时候，slow节点正好指向了倒数第K个节点的位置

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */

/**
 * @param {ListNode} head
 * @param {number} k
 * @return {number}
 */
var kthToLast = function (head, k) {
    let fast = slow = head;
    for (let i = 0; i < k; i++)fast = fast.next;
    while (fast) {
        fast = fast.next;
        slow = slow.next;
    }
    return slow.val;
};

时间复杂度：O(n)

空间复杂度：O(1)

【876】链表的中间节点

题目描述

题目链接

解法

通过两个指针，一个快指针fast，一个慢指针slow，两个指针同时从链表头结点开始遍历。

快指针每次向后移动两个节点，慢指针每次向后移动一个节点，当快指针到达链表末尾的时候，慢指针就恰好在链表的中间位置

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var middleNode = function (head) {
    if (!(head && head.next)) return head;
    let fast = slow = head;
    while (fast && fast.next) {
        fast = fast.next.next;
        slow = slow.next;
    }
    return slow;
};

时间复杂度：O(n)

空间复杂度：O(1)

如果不要求空间复杂度为O(1)的话，还可以使用一个O(n)的空间复杂度的解法，将链表遍历依次，用数组将每个节点存储下来，然后根据数组的长度直接确定中间节点的索引位置

【61】旋转链表

题目描述

题目链接

解法

双指针法

将一个链表的每个节点都向后移动K个位置，实际上就是将链表最后的K个节点形成的子链表，移动到链表的开头位置

例如示例中

1->2->3->4->5->NULL, k = 2

实际就是将后面的两个节点4->5，移动到链表的开头位置，变成

4->5->1->2->3->NULL

因此，可以通过两个指针fast和slow，首先先让fast指针往前移动k个节点，然后再让fast和slow一起同时往后移动，这样当fast指针刚好遍历到链表末尾的时候，slow指针就正好停在了倒数第K个节点的位置，然后只需要将slow节点后面的子链表作为原本链表的开头即可

上面的分析是基于k < n的情况的，也即移动的位置是小于节点的个数的。当k>=n的时候，需要让链表可以重新回到头结点，重新开始遍历，因此fast指针 先走K个位置，应该按照下面的代码来执行

  let fast = slow = head, prev = tail = null;
  for (let i = 0; i < k; i++)fast = fast.next || head;

同时还应该考虑到，既然K可能大于n，则fast指针在先走了k个节点之后，其与slow指针之间的位置关系就有两种情况

第一种就是正常情况，fast指针在slow指针前面。这种情况下，只需要按照上面的分析过程，在fast移动到链表末尾后，将slow指向的节点作为新的头结点。这里还要注意，将slow后面的节点形成的子链表移动到原本链表开头的时候，要讲前一个链表的连接断开，否则链表会形成一个环

let prevHead = head;
head = slow;
tail.next = prevHead;
prev.next = null;
return head;

第二种情况是fast指针恰好停留在了slow指针所在的位置，也即头结点，这是因为k % n === 0的情况造成的，这种情况下，就相当于链表中每个节点向后移动的位置是0，因此最后的结果就是原本的链表，此时直接返回原本的头结点即可

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} k
 * @return {ListNode}
 */
var rotateRight = function (head, k) {
    if (!head || k === 0) return head;
    let fast = slow = head, prev = tail = null;
    for (let i = 0; i < k; i++)fast = fast.next || head;
      if (slow === fast) return head;
    while (fast) {
        prev = slow;
        tail = fast;
        fast = fast.next;
        slow = slow.next;
    }
    let prevHead = head;
    head = slow;
    tail.next = prevHead;
    prev.next = null;
    return head;
};

时间复杂度：O(n)

空间复杂度：O(1)

解法

先用一个Map将所有节点存入到其中，然后依次取出来，再连接上next和random指针即可

/**
 * // Definition for a Node.
 * function Node(val, next, random) {
 *    this.val = val;
 *    this.next = next;
 *    this.random = random;
 * };
 */

/**
 * @param {Node} head
 * @return {Node}
 */
var copyRandomList = function (head) {
    if (head == null) return head;
    let current = head;
    const map = new Map()
    while (current) {
        const newNode = new Node(current.val, null, null);
        map.set(current, newNode)
        current = current.next;
    }
    current = head;
    while (current) {
        if (map.has(current)) {
            let e = map.get(current)
            const next = map.get(current.next)
            const random = map.get(current.random)
            next && (e.next = next);
            random && (e.random = random)
        }
        current = current.next;
    }
    return map.get(head)
};

【剑指offer-18】删除链表的节点

题目描述

题目链接

解法

对链表进行遍历，将当前节点和前一个节点保存下来，找出要删除的指定节点后，将前一个节点的next指针指向当前节点的后继节点，同时将当前节点的next指针断开即可

/**

 * function ListNode(val) {
 *     this.val = val;

 * }

/**
 * @param {ListNode} head
 * @param {number} val
 * @return {ListNode}

var deleteNode = function (head, val) {
    if (!head || (val == null)) return head;
    let current = head;
    let prev = null;
    while (current) {
        if (current.val === val) {
            if (!prev) {
                head = head.next;
            } else {
                prev.next = current.next;
            }
            current.next = null;
            break;
        } else {
            prev = current;
            current = current.next
        };
    }
    return head;
};

解法

通过对链表进行遍历，如果当前节点的值和指定的值相等，则当前节点需要被删除。

在删除节点的时候，通过前一个节点的指针prev，将prev.next = current.next和current.next来将当前节点链表中移除

应该注意的是，因为这个题里面，要将链表中所有节点值等于指定值的节点都删除，因此在删除了前面的节点后，还需要继续向后遍历去删除其他需要删除的节点，这个时候，只需要将current = next来继续向后遍历即可，prev指针应该停留在原来的节点不动，否则对于连续多个节点都是需要删除的节点的时候，就会有一些节点被跳过。

/**

 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @param {number} val
 * @return {ListNode}
 */
var removeElements = function (head, val) {
    if (!head || val == null) return head;
    let current = head;
    let prev = null;
    while (current) {
        if (current.val === val) {
            const next = current.next;
            if (!prev) head = next;
            else prev.next = next;
            current.next = null;
            current = next;
        } else {
            prev = current;
            current = current.next;
        }
    }
    return head;
};

    let prev = null;

        current.next = prev;
        prev = current;

对于两个链表，对每个链表都进行分段处理。两个链表的公共子链表部分即为C，第一个链表的剩余的非公共部分即为A，第二个链表的非公共部分记为B 。第一个链表可以记为A+C，第二个链表可以记为B+C。

 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var removeZeroSumSublists = function (head) {
    if (!head) return head;
    let current = head;
    let prev = null;
    while (current) {
        if (current.val === 0) {
            if (!prev) head = current.next;
            else prev.next = current.next;
            const next = current.next;
            current.next = null;
            current = next;
            continue;
        }
        let sum = current.val;
        let remain = current.next;
        let isDeleted = false; // 在下面的remian的循环中是否找到并删除了总和值为零的子链表段
        while (remain) {
            sum += remain.val;
            if (sum === 0) {
                if (!prev) head = remain.next;
                else prev.next = remain.next;
                current = remain.next;
                remain.next = null;
                isDeleted = true;
                break;
            }
            remain = remain.next;
        }
        if (!isDeleted) { // 删除了链表段的时候，prev指针停在原位，current指针移到 remain.next
            prev = current;
            current = current.next;
        }

    }
    return head;
};

Map存储总和

通过两次遍历链表，第一次遍历的时候，通过一个Map，将每个节点与其前面节点的总和值作为key，当前节点作为值value存储起来

因为如果两个节点之间的节点总和值为0的话，则两个节点在计算当前节点与其前面节点的总和值的时候，得到的sum应该是相等的，在存到map的时候，就会覆盖掉前面的节点。

而在第二次遍历的时候，为了找到哪些节点之间的总和值为0，就可以看一下当前节点current与其前面节点的总和值sum为键，对应的节点value是哪一个，如果current与value两个节点是同一个节点，则说明当前节点与其后面的所有节点之间的总和值都不是0，则只需要继续遍历下一个节点即可。而如果两个节点不是同一个节点，则说明这两个节点之间的总和值为0，则需要将这两个节点之间的子链表删除掉，也即是执行current.next = map.get(sum).next

/**
 * Definition for singly-linked list.
 * function ListNode(val) {
 *     this.val = val;
 *     this.next = null;
 * }
 */
/**
 * @param {ListNode} head
 * @return {ListNode}
 */
var removeZeroSumSublists = function (head) {
    if (!head) return head;
    let newHead = new ListNode(-1);
    newHead.next = head;
    let sum = 0;
    const map = new Map();
    let current = newHead;
    while (current) {
        sum += current.val;
        map.set(sum, current);
        current = current.next;
    }
    sum = 0;
    current = newHead;
    while (current) {
        sum += current.val;
        current.next = map.get(sum).next;
        current = current.next;
    }
    return newHead.next;

};
